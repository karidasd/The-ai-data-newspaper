# AI & Data Science Newsletter Bot 🤖📰

Αυτό το αποθετήριο δημιουργεί αυτόματα μια εβδομαδιαία εφημερίδα με τα τελευταία νέα από τον χώρο της Τεχνητής Νοημοσύνης και της Επιστήμης Δεδομένων.

Χρησιμοποιεί **Python** και **GitHub Actions** για να συλλέγει άρθρα μέσω RSS feeds κάθε Παρασκευή, δημιουργώντας αυτόματα ένα νέο αρχείο Markdown!

## Πώς Λειτουργεί
1. Ένα GitHub Action εκτελείται αυτόματα (ή χειροκίνητα).
2. Τρέχει το `generate_newsletter.py`.
3. Το script διαβάζει RSS feeds από:
   - MIT News (AI)
   - Towards Data Science
   - KDNuggets
   - Google AI Blog
4. Δημιουργεί ένα νέο τεύχος στο φάκελο `issues/` και ενημερώνει αυτό το README.
5. Κάνει commit και push αυτόματα!
